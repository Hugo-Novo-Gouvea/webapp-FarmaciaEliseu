- O item selecionado no campo select em "Coluna" determina qual a coluna onde a pesquisa será feita
- "Pesquisa" é um input que filtra no campo apenas os item onde tem presente o valor digitado referente a coluna escolhida em "Coluna" apresentando itens onde o valor está presente em qualquer parte do texto
- Após entrar na tela a opção inicial no campo coluna é "Codigo de Barras"

Tabela: dbo.produtos
-------------------------------------------------
produtosID        int               PK, NOT NULL
descricao         nvarchar(200)     NOT NULL
unidadeMedida     nvarchar(200)     NULL
precoCompra       decimal(19,4)     NULL
precoVenda        decimal(19,4)     NOT NULL
localizacao       nvarchar(200)     NULL
laboratorio       nvarchar(200)     NULL
principio         nvarchar(200)     NULL
generico          nvarchar(3)       NOT NULL
codigoProduto     nvarchar(9)       NULL
codigoBarras      nvarchar(50)      NULL
dataCadastro      datetime2(0)      NULL
dataUltimoRegistro datetime2(0)     NULL
deletado          bit               NULL

---EDITAR---

	- retorna um modal preenchido com as informações de produtos e com a capacidade de ser alterados
		"codigoBarras", 
		"codigoProduto", 
		"descricao", 
		"unidadeMedida", 
		"precoCompra", 
		"precoVenda", 
		"localizacao", 
		"laboratorio", 
		"principio", 
		"generico"

	- regras de cada campo:

		Codigo de Barras: fornece valor para o campo "codigoBarras" no dbo.produtos
		- tipo de campo é number
		- Não pode ser nulo

		codigoProduto: fornece valor para o campo "codigoProduto" no dbo.produtos
		- mesmo recebendo letras minusculas deve ser salvo no banco com todas as letras maiusculas
		- pode ser nulo
		- caso seja nulo o banco recebe o valor "000000000"

		Descrição: fornece valor para o campo "descricao" no dbo.produtos
		- mesmo recebendo letras minusculas deve ser salvo no banco com todas as letras maiusculas
		- Não pode ser nulo

		Unidade de Medida: fornece valor para o campo "unidadeMedida" no dbo.produtos
		- mascara apenas numeros
		- pode ser nulo
		- caso seja nulo o banco recebe o valor 0

		Preço de Compra: fornece valor para o campo "precoCompra" no dbo.produtos
		- mascara apenas numeros
		- Não pode ser nulo
		- mostrar o valor sempre com dois zeros depois da virgula

		Preço de Venda: fornece valor para o campo "precoVenda" no dbo.produtos
		- mascara apenas numeros
		- Não pode ser nulo
		- mostrar o valor sempre com dois zeros depois da virgula

		Localização: fornece valor para o campo "localizacao" no dbo.produtos
		- mesmo recebendo letras minusculas deve ser salvo no banco com todas as letras maiusculas
		- pode ser nulo
		- caso seja nulo o banco recebe o valor "NAO INFORMADO"

		Laboratorio: fornece valor para o campo "laboratorio" no dbo.produtos
		- mesmo recebendo letras minusculas deve ser salvo no banco com todas as letras maiusculas
		- pode ser nulo
		- caso seja nulo o banco recebe o valor "NAO INFORMADO"

		principio: fornece valor para o campo "principio" no dbo.produtos
		- mesmo recebendo letras minusculas deve ser salvo no banco com todas as letras maiusculas
		- pode ser nulo
		- caso seja nulo o banco recebe o valor "NAO INFORMADO"

		generico: fornece valor para o campo "generico" no dbo.produtos
		- campo select box com valores "Selecione um valor", "SIM", "NAO", permitindo registro no banco apenas se valor diferente de "Selecione um valor"
		- não pode ser nulo

	- ao ser feito o registro no banco é atualizado o campo "dataUltimoRegistro" com a data e horário atual
	- botão cancelar para cancelar a operação
	- botão salvar para realizar a operação
