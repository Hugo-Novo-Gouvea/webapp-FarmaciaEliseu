Regras do CRUD

Banco: farmaciaEliseu
Tabela: dbo.funcionarios
-------------------------------------------------
funcionariosID      int             PK, NOT NULL
nome                nvarchar(200)   NOT NULL
dataCadastro        datetime2(0)    NOT NULL
dataUltimoRegistro  datetime2(0)    NOT NULL
deletado            bit             NOT NULL

---VISUALIZAR---

	- retorna um modal com as informações do funcionario selecionado preenchidos e sem a capacidade de ser alterados
		"nome",

---EDITAR---

	- retorna um modal preenchido com as informações de funcionarios e com a capacidade de ser alterados
		"nome"

	- regras de cada campo:

		Nome: fornece valor para o campo "nome" no dbo.clientes
		- mesmo recebendo letras minusculas deve ser salvo no banco com todas as letras maiusculas
		- Não pode ser nulo

	- ao ser feito o registro no banco é atualizado o campo "dataUltimoRegistro" com a data e horário atual
	- botão cancelar para cancelar a operação
	- botão salvar para realizar a operação

---NOVO---

	- retorna um modal em branco com as informações de funcionarios e com a capacidade de ser alterados (Sendo endereco a concatenação de "logradouro, numero, bairro")
		"nome"

	- regras de cada campo:

		Nome: fornece valor para o campo "nome" no dbo.clientes
		- mesmo recebendo letras minusculas deve ser salvo no banco com todas as letras maiusculas
		- Não pode ser nulo

	- ao ser feito o registro no banco é atualizado os campos "dataCadastro" e "dataUltimoRegistro" com a data e horário atual
	- botão cancelar para cancelar a operação
	- botão salvar para realizar a operação

---EXCLUIR---

- soft delete
- altera o deletado do item de false para true
